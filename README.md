# Hardware Labs

This repository contains electronics, circuit builds, and embedded system experiments.

## Focus Areas

- Breadboard circuits
- Sensors and components
- Motor control
- Prototyping
- Embedded hardware practice

## Contents

- Lab builds
- Circuit diagrams
- Wiring notes
- Test results
- Observations and troubleshooting notes

## Example Projects

- LED and resistor circuits
- Button input systems
- Temperature sensor builds
- Servo and motor control labs

## Purpose

The goal of this repository is to develop practical hardware and electronics skills through hands-on experimentation.

## Notes

Projects may include both simple circuit exercises and more integrated system builds.
